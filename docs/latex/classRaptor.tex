\doxysubsection{Detailed Description}
Implements the RAPTOR algorithm for finding Pareto-\/optimal journeys. 

The \doxylink{classRaptor}{Raptor} class provides methods to set a query, find Pareto-\/optimal journeys, and print journey steps. It uses various data structures to store information about agencies, calendars, stops, routes, trips, and stop times.
