\doxysubsection{Detailed Description}
Represents a stop in the GTFS data. 

This class inherits from \mbox{\hyperlink{classGTFSObject}{GTFSObject}} and manages stop time and route information for a specific stop. It provides methods for adding stop time and route IDs, retrieving sorted data, and defining custom sorting mechanisms.
