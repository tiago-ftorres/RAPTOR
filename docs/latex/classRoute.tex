\doxysubsection{Detailed Description}
Represents a route in the GTFS data. 

This class inherits from \doxylink{classGTFSObject}{GTFSObject} and manages trip and stop information for a specific route. It provides methods for adding trip and stop IDs, retrieving sorted data, and defining custom sorting mechanisms.
