\doxysubsection{Detailed Description}
Represents a trip in the GTFS data. 

This class inherits from \doxylink{classGTFSObject}{GTFSObject} and manages stop time information for a specific trip. It provides methods for adding stop time keys, retrieving sorted data, and defining custom sorting mechanisms.
